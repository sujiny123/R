# 기본문법
## 도움말 : 함수나 명령어에 대한 설명과 사용법을 확인
help(log)<br>
?log

## 예약어에 대한 도움말을 볼 경우 앞뒤에 쌍따옴표
help("for")<br>
example(lm)

## 여러줄 주석 처리 : 영역 지정 후 ctrl + shift + C

## 산술연산자자 : +,-,*,/(실수몫),%%(정수몫),%/%(나머지),^,**(거듭제곱)

(11+54-13)*76/9<br>
9/3<br>
8/3<br>
8%%3<br>
8%/%3<br>
